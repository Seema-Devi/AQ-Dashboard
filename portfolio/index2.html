GDDA713 Capstone Project – NZSE

1. Project Overview

This project develops an AI‑powered forecasting system to analyse and predict short‑term air quality (PM2.5, NO₂, AQI) in Auckland City Centre. The system integrates air quality, meteorological, traffic, and pedestrian datasets to support Auckland Council with proactive environmental planning.

3. Data Cleaning Methods

  • Missing Value Handling: Forward fill, backward fill, median imputation.
  • Datetime Standardisation: Unified timestamp formatting.
  • Dataset Merging: Timestamp‑aligned merging of all datasets.
  • Duplicate & Invalid Removal: Removal of duplicates and negative values.
  • Numeric Validation: Conversion to numeric types.
  • Outlier Detection: Boxplots and statistical summaries.
  • Column Standardisation: Clean variable names.
  • Time‑Series Continuity: No row deletions.
