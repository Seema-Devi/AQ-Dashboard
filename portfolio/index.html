AI-Driven Air Quality Forecasting Dashboard

A capstone project focused on analysing Auckland air quality data and developing machine learning models to forecast future pollution levels using Random Forest, XGBoost, and LSTM.

Project Overview

This project develops an interactive forecasting dashboard for air quality monitoring. The system supports data cleaning, exploratory analysis, feature engineering, model optimisation, and 24-hour forecasting of selected air quality parameters.

GitHub Repository Live Dashboard

Project Objectives

  • Analyse air pollution and meteorological trends.
  • Apply feature engineering using lag, rolling, and time-based features.
  • Train and optimise Random Forest, XGBoost, and LSTM models.
  • Generate 24-hour forecasts for selected air quality parameters.
  • Present insights using an interactive Streamlit dashboard.

Technologies Used

Model Development

The modelling stage compares traditional machine learning models with deep learning. Random Forest and XGBoost are used as baseline and optimised models, while LSTM is used for time-series forecasting because it can learn sequential pollution patterns.

Model Purpose Result
Random Forest Baseline machine learning model Used for comparison
XGBoost Optimised ensemble model Used for improved prediction
LSTM Time-series forecasting model Used for 24-hour forecasting

Challenges and Improvements

  • Handling missing values and inconsistent time-series records.
  • Improving EDA using environmental visualisations such as wind and seasonal analysis.
  • Applying model optimisation and cross-validation to improve reliability.
  • Developing recursive 24-hour forecasting for future pollution prediction.

Final Outcome

The final outcome is an interactive AI-based forecasting dashboard that supports air quality analysis, model comparison, optimisation, and next 24-hour prediction. The project demonstrates how machine learning can support environmental monitoring and data-driven decision-making.
